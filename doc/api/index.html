Libraries

csp
CSP is a library to easily solve any kind of CSP some could come up with. To define a Constraint Satisfaction Problem (CSP) use the CSP constructor, to initialize the needed CSPVariable and CSPConstraint to define constraints between them.
csp_constraints
This Library provides predefined CSPConstraint that are ready to use. To define custom made Constraints write a class implementing CSPConstraint.
csp_solvers
This Library provides predefined CSPConstraint that are ready to use. To define custom made Constraints write a class implementing CSPConstraint.
