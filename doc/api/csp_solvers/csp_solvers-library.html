csp_solvers

csp_solvers library

This Library provides predefined CSPConstraint that are ready to use. To define custom made Constraints write a class implementing CSPConstraint.

Classes

MultipleAssignmentSolver
The MultipleAssignmentSolver class implements a backtracking algorithm designed to solve Constraint Satisfaction Problems (CSPs) by assigning multiple values to each variable.
SingleAssignmentSolver
The SingleAssignmentSolver class provides a backtracking algorithm to solve Constraint Satisfaction Problems (CSPs) by assigning a single value to each variable.
