csp

csp library

CSP is a library to easily solve any kind of CSP some could come up with. To define a Constraint Satisfaction Problem (CSP) use the CSP constructor, to initialize the needed CSPVariable and CSPConstraint to define constraints between them.

Classes

ArcPruner
ArcPruner is an interface to define pruner classes, used to help find inconsistencies and arc incompatiblilities between Variables inside of a CSP.
CSP<T>
The csp_model.dart file defines the fundamental structure of the Constraint Satisfaction Problem (CSP) which is essential for running a solver on top of it.
